## FST (openfst? morfologik?)

```
for (StateIterator< FST<A>> siter(fst); 
     !siter.Done();
     siter.Next() {
     StateId s = siter.Value();
     if (fst.Final(s) != Weight::Zero()) {
          //do stuff
     }
}
```

`Weight::Zero()` applies with unweighted FSTs
