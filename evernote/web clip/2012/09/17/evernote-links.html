[How to clear font caches in Leopard Macworld](http://www.macworld.com/article/1139383/fontcacheclear.html)

If you want to remove the font cache for all users, use this command, and provide your admin password when asked: sudo atsutil databases -remove

Once you’ve cleared the caches, you should stop and restart the ATS server with these commands:

$ atsutil server -shutdown

$ atsutil server -ping
